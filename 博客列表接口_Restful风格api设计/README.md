## Getting Started
### Clone Project
你可以在本機直接使用 git 指令 clone 此專案並執行。

```
git clone https://github.com/uchpython12/FastAPI
```

```
python /main.py
```

running locally! Your app should now be running on [localhost:8888](http://localhost:8888/) .
